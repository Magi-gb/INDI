# INDI
Projecte de INDI 2026
Aquest projecte es basa en la creació d'un joc funcional on el jugador mou un Morty al voltant d'un laberint mentres agafa monedes, i es perseguit per un fantsama. 
EL projecte esta escrit en llengiatje C i C++, basat en Qt i GLM.
